GitHub basics

GitHub is where we store and share code. Think of it like a shared drive for projects, with history so we can see what changed and when.

What’s a repo?

A repository (or “repo”) is a project folder that lives on GitHub. It contains all the files for that project, plus the history of changes. When you clone a repo, you copy it to your computer so you can work on it.

Why we use it

  • Everyone works from the same place and can see the latest version.
  • Changes are tracked, so we can see who did what and when.
  • We can collaborate without overwriting each other’s work.

Create a GitHub account

If you don’t have an account yet, go to github.com/join and sign up. It’s free. Once you’re in, you’ll be able to clone repos and, when you’re ready, push your own changes.
