# Avatars

Add your avatar image here to show on the Agents page.

1. Add your image file (e.g. `jane-doe.jpg` or `jane-doe.png`).
2. In `src/data/contributors.json`, set your entry's `avatarUrl` to `"/avatars/your-filename.jpg"` (use the path starting with `/avatars/` and your actual filename).

Keep filenames lowercase and use dashes instead of spaces (e.g. `jane-doe.jpg`).
