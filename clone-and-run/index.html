Clone and run

Follow these steps to copy this workshop to your computer and run it in your browser. This is the same flow you’ll use for other projects: clone, open in Cursor, install, run.

1

Clone the repo

Open a terminal (or use the one inside Cursor: Terminal → New Terminal). Then run the command below. Replace your-username with the GitHub username or org that owns this repo.

git clone https://github.com/your-username/cursor-workshop.git

This creates a folder called cursor-workshop with all the project files.

2

Open the folder in Cursor

In Cursor: File → Open Folder (or Cmd+O / Ctrl+O). Choose the cursor-workshop folder you just cloned.

3

Install dependencies

In the terminal (you can use the one at the bottom of Cursor), run:

npm install

This downloads the packages the project needs. Wait until it finishes.

4

Start the dev server

In the same terminal, run:

npm run dev

The site will start. You should see a message like “Local: http://localhost:4321” (or another port). Open that link in your browser – you’re now viewing the workshop from your own computer.

You’ve just run a project on localhost. Next: Explore Cursor to learn the interface, or jump to the Checklist to track your progress.
