Welcome to the Cursor Workshop

This workshop helps you get started with Cursor and GitHub. By the end, you’ll have cloned this repo, opened it in Cursor, run it on your computer, and explored the basics of both tools.

What you’ll learn

  • What GitHub is and why we use it
  • How to clone a repo and run it locally
  • How to find your way around Cursor (sidebar, command palette, chat)
  • A simple checklist so you can track your progress

What you need

  • Cursor – Download Cursor if you haven’t already.
  • GitHub account – You’ll create one in the next section if you don’t have it yet.

Ready? Start with GitHub basics, then follow Clone and run to get this workshop running on your machine.
