Explore Cursor

Cursor is a code editor with AI built in. Here’s a quick tour of the main areas. Add your own descriptions and screenshots later by editing this page and dropping images into public/images/.

Sidebar and file tree

The left sidebar shows your project’s files and folders. Click a file to open it. You can collapse or expand folders to find what you need.

[Screenshot: Cursor sidebar / file tree – add public/images/cursor-sidebar.png]

Command Palette

The Command Palette lets you run commands and jump to settings without using the menu. Try it: Press Cmd+Shift+P (Mac) or Ctrl+Shift+P (Windows/Linux), then type “Command Palette” or any command name.

[Screenshot: Command Palette – add public/images/command-palette.png]

Cursor Chat

Cursor’s chat lets you ask questions about your code, get explanations, or request changes. Open it from the sidebar or with the shortcut shown in the Cursor menu. You can reference specific files so the AI has context.

[Screenshot: Cursor chat – add public/images/cursor-chat.png]

Composer

Composer is where you can request multi-file edits or new features. It’s useful when you want to change several files at once or generate something new. You’ll find it in the Cursor UI when you’re ready to try it.

[Screenshot: Composer – add public/images/composer.png]
